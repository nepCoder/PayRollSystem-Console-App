# PayRollSystem-Console-App
PayRollSystem Console App using C#.NET6.

>Learnig Objectives
- Object Oriented Programming with C#
- Class and Objects
- Inheritance
- Class Fields
- Properties (with both default and manual getter, setter methods)
- Methods
- Abstraction
- Method Override
- File Handling
and --many more--
